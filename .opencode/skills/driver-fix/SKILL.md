## What I do

- Diagnose driver failures using Device Manager and system logs
- Guide through driver reinstallation and rollback procedures
- Check Windows Update and manufacturer driver sources
- Document fixes in the project's existing markdown format

## When to use me

Use this when the user reports hardware not working, driver errors in Device Manager, or issues after Windows updates. Always check existing documentation first before suggesting fixes.
